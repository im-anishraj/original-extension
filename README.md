# ✂️ Disable Copy Paste Chrome Extension

This Chrome extension automatically blocks copy-paste and right-click functionality on websites when toggled on. It also gives you full control with a site-specific toggle and persistent memory.

---

## 🚀 Features

- 🚫 Disables copy, paste, text selection, and right-click on activated websites
- 탭 Blocks opening new tabs when active
- 🔘 Toggle functionality per website via extension popup
- 🧠 Remembers your preferences using Chrome storage
- ⚡ Lightweight and fast
- 🔒 Open source and privacy-friendly (no data is collected)

---

## 🛠 How It Works

- The extension checks if the current website is in your **active list**.
- If activated, it prevents common JavaScript events to block:
  - `copy`, `paste`, `cut`
  - `right-click` (`contextmenu`)
  - `text selection`
  - It also blocks keyboard shortcuts like `Ctrl+C`, `Ctrl+V`, `Ctrl+X`, `Ctrl+A`.
- You can click the extension icon to **enable or disable** blocking for the current site.

---

## 📦 Quick Install Options

1. Download the `.zip` from Releases
2. Extract the ZIP
3. Open `chrome://extensions/`
4. Enable **Developer mode**
5. Click **Load unpacked** and select the extracted folder
